Software Licenses

A software license is an agreement between a user and the owner of a software defines what all user can do with that software. Using it in any other way would amonunt to an infringement of the software license.

There are two common categories of software under copyright law to grant hte license specific rights.

  • Propietary Rights
  • Free and open source software
Open Source Software Movement

In 1983, Richard Stallman, launched the GNU project to write a complete operating system making its source code freely available. GNU software are free not in terms of price, but in terms of accessiblity.

GNU project is a free and mass collaboration project, Its aim is to give rights to a user to run, share, copy, distribute, study, and modigy the software.

In 1985, the Free Software Foundation was formed for funding the GNU project. Four rules define a free software. They are:

  1. Run the program for any purpose.
  2. Identify the program according to your own speed.
  3. Redistribute copies
  4. To destribute copies of your modified copies to others.

GNU project joined Linux in 1990. The free operating system was finally created and was named as GNULinux.
