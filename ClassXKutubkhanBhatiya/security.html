Security of Information

Security of information ensures prevention of unauthorized access to information and corruption of content. Common threats to security of information include unauthorized access and by malicious software such as virus, worms, and trojan horses. While the former steal the information, the later cause corruption or loss of information.

Computer hacking refers to gaining unauthorized access to a computer or network resources. A person does hacking is called a hacker or a cracker. A hacker is a person who gains unauthorized access toa computer for the porpose of fixing flaws whithin the system. Whereas, a cracker id one who gains unauthorized access to a computer for the purpose of stealing or corrupting data.

The following are often used to prevent the security and intergrity of information:

Password  A password is a secret sequence of characters that enables a user to control access of a resource such as a computer, a program, or a file. To prevent unauthorized access we should not reveal our password to anyone, and should also make a strong password.

Antivirus software  It detects and removes virus and other mailicious software from a computer. It needs to be updated regularly to amintain its effectiveness.

Firewall  A firewall is a security system designed to protect a computer from hacker attacks. Users are generally advised to install an anti-spyware software and not click on e-mails received from unknown sourcesto avoid falling prey to such attacks.

Secure Data Transmission

The information security system in e-commerce should meet the following requirements:

  • Privacy  No unauthorized person should be able to access the information.
  • Authentication of sender and receiver  Both sender and receipent msut prove their identities to each other.
  • Intergrity and non-repudation  The information must not be tampered with and there should be a proof that the information is intact.

Secure Sockets Layer(SSL)

SSL, or Secure Sockets Layer, is an encryption-based Internet security protocol. It was first developed by Netscape in 1995 for the purpose of ensuring privacy, authentication, and data integrity in Internet communications. SSL is the predecessor to the modern TLS encryption used today.

Public and Private key encryption  Privacy of inf is handled by encryption. Encryption is the process of modifying data so that unauthorized users cannot read it. Decryption is the process of converting encrypted data back to human-readable form. This method solves the problem by creating a set of two different keys called a key pair. While one of the two keys is used to encrypt data the other can be used to decrypt data. The two keys in a key pair are as follow.

  • Private key,  Only the owner knows the private key.
  • Public key  It is distributed to any user who requests it.

Digital certificate

A Digital Certificate is an electronic "password" that allows a person, organizaion to exchange data securely over the Internet using the public key infrastructure (PKI). Digital Certificate is also known as a public key certificate or identity certificate.
The digital certificate typically includes the following information:

  • The name of the owner, its email address, and the URL of the web server.
  • The owners public key.
  • The name of the CA that issues the certificate.
  • A serial number.
  • The validity period of the certificate.
  • The digital signature of the CA.
